# Vendor-Performance-Analysis
# 📦 Inventory Data Pipeline & Analysis

A Python-based data pipeline for ingesting, processing, and analyzing inventory data using MySQL and Pandas.

---

## 🚀 Overview

This project automates:
- Loading CSV data into a MySQL database  
- Performing exploratory data analysis (EDA)  
- Analyzing vendor performance for business insights  

---

## ⚙️ Tech Stack

- Python  
- Pandas  
- NumPy  
- SQLAlchemy  
- MySQL  
- Jupyter Notebook

## 🔄 Workflow

### 1. Data Ingestion
- Reads CSV files from `data/` folder  
- Cleans column names  
- Handles missing and infinite values  
- Loads data into MySQL tables  

### 2. Exploratory Data Analysis
- Connects to database  
- Explores tables and records  
- Performs grouping and filtering  

### 3. Vendor Analysis
- Evaluates vendor performance  
- Identifies cost and efficiency patterns  
- Generates business insights
